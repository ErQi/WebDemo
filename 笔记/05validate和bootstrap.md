# bootstrap #
Bootstrap 是基于 html、css、javascript的的前端框架.
他是一款响应时框架,可以根绝屏幕大小的不同,动态的调整界面的布局.
核心是通过栅格系统,将屏幕分为12等分,在布局时决定在不同屏幕下所占不同等分来进行不同的布局.
更将一些js框架将基本按钮整合成不同菜单,在不同屏幕下动态调整其显示效果.
[使用教程](http://v3.bootcss.com/)
